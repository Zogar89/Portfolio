Meli Core Dumper

Un core multitenant read-only para transformar datos de Mercado Libre en metricas confiables, auditables y listas para insights.

No alcanza con traer datos. Hay que saber si se pueden creer.

Contexto

Un seller necesita entender que paso con cada publicacion: visitas, ventas, ads, preguntas, reviews, stock, precio y conversion. Las fuentes son parciales, rate-limited y cambian en el tiempo.

Construccion

Disene una plataforma read-only con OAuth, workers, scheduler, checkpoints, metric sync runs, snapshots diarios y agregados por publicacion.

Decision clave

El sistema separa facts, snapshots, aggregates, readiness e insights. Si la cobertura no alcanza, explica el motivo en vez de inventar una conclusion.

Una arquitectura para sincronizar, auditar y explicar.

La prioridad fue que cada dato tenga trazabilidad operativa. Runs, steps, checkpoints y eventos permiten entender que fuente fallo, que rango se proceso y que tan confiable es el resultado.

  • Multitenancy and OAuth2 Cuentas conectadas con tokens cifrados, scopes controlados y separacion estricta por tenant.
  • Background jobs ARQ, Redis y scheduler para sincronizaciones largas, reintentos, backfills y procesos auditables.
  • Observability by design Admin de sync runs, eventos operativos, logs seguros, cobertura diaria y diagnosticos sin filtrar secretos.
  • Responsible insights El LLM, cuando aparece, narra evidencia estructurada. No calcula facts ni toma atajos causales.

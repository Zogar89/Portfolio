Case study

Meli Core Dumper

Una plataforma de inteligencia de negocios para vendedores de Mercado Libre que detecta patrones, anomalias y tendencias en sus publicaciones.

Ver repo Mas proyectos
Rol Arquitectura, backend, datos, admin
Dominio Mercado Libre, BI, estadistica
Stack FastAPI, PostgreSQL, Redis, ARQ

Arranco como un dump. Se convirtio en inteligencia dentro del producto.

Origen

La idea inicial era descargar toda la informacion de la operacion, exportarla a Excel y analizarla afuera, como un backup o volcado de base de datos.

Giro de producto

La pregunta clave fue simple: si ya tenemos los datos, por que salir del sistema para analizarlos. El dumper paso a ser una plataforma de decision.

Valor actual

En vez de obligar al vendedor a mirar dashboards, el sistema busca la informacion importante: patrones, anomalias, tendencias y senales para actuar.

Business intelligence sobre publicaciones reales, no solo reporting.

La plataforma combina sincronizacion operativa con matematica estadistica aplicada para detectar cambios relevantes en todas las publicaciones de un vendedor. El objetivo es liberar tiempo: menos exploracion manual, mas senales listas para decidir.

  • Statistical pattern detection Analisis de comportamiento historico para encontrar cambios que una lectura manual de dashboard puede pasar por alto.
  • Anomaly and trend analysis Deteccion de variaciones, caidas, subas y movimientos raros en visitas, ventas, conversion, ads y actividad por publicacion.
  • Seller-wide intelligence Vision sobre todas las publicaciones del vendedor, no una pantalla aislada por metrica.
  • Operational data foundation OAuth, workers, scheduler, checkpoints, snapshots, agregados diarios y observabilidad para sostener el analisis.

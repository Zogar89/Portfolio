Stock Central

Una plataforma comunitaria para centralizar stock de filamentos 3D y evitar perseguir distribuidores por cada color, material o presentacion.

Nacio de una friccion real del mercado 3D.

Origen

Imprimo y vendo impresiones 3D en Mercado Libre. El filamento tiene muchas variantes: colores, materiales, presentaciones y marcas.

Dolor

Para un comprador mayorista era desgastante preguntar a cada distribuidor si habia entrado una variante. Con desabastecimiento e importaciones trabadas, ese problema se volvio cotidiano.

Producto

Algunos distribuidores empezaron a publicar stock online. Centralice esas fuentes y sume busqueda, ayuda para armar pedidos y avisos sobre variantes necesarias.

Backend en jobs. Runtime estatico.

La clave tecnica es que todo se sostiene sobre GitHub Pages. No hay servidor propio en runtime: la logica de backend vive encapsulada en jobs de GitHub que actualizan el sitio 4 veces al dia.

  • Multi-source ingestion Toma stocks desde Google Sheets publicados como CSV y desde HTML de proveedores, con conectores separados por fuente.
  • Official color and image data Lee el catalogo, sitemap y fichas de Grilon3 para enriquecer productos con URL oficial, imagen, Pantone, SKU y EAN.
  • Integrity checks before publish Valida schema, errores de fuentes, caidas sospechosas de productos y bajas bruscas de stock antes de publicar.
  • Community-first infrastructure Mantiene costo operativo muy bajo: GitHub Actions para actualizar, GitHub Pages para servir, cero backend permanente.

Captura de datos sin backend permanente.

Google Sheets

Los proveedores que publican stock en planillas se leen como export CSV de Google Sheets. El parser detecta columnas de producto, stock y marca, repara mojibake y descarta filas que no son producto.

HTML de proveedores

Filamentos3D se captura con httpx, BeautifulSoup y lxml: primero tablas de stock y despues paginas de categoria y detalle para imagenes, SKUs y links de producto.

Catalogo Grilon3

Grilon3 usa requests y parsers HTML propios sobre catalogo, sitemap y fichas WooCommerce. De ahi salen imagenes oficiales, Pantone, SKU, EAN y URLs del fabricante.
